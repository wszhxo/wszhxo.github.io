浅克隆

我们都知道一个对象是存储在堆内存中的,对象的引用来源于此,基本数据类型不存在这个问题

p1和p2都指向同一个Person对象,当p1和p2 其中任意一个修改了Person的值那么Person对象就会发生改变,所以直接赋值就是浅克隆.
